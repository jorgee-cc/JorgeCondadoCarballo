## 🎯 Objetivo del Proyecto
El objetivo de este proyecto es construir una carta de presentación digital interactiva, rápida y escalable. Está diseñado con una arquitectura modular basada en componentes, priorizando la accesibilidad web, la experiencia de usuario (UX) y un rendimiento ultrarrápido.

## 🛠️ Stack Tecnológico
Este proyecto ha sido construido utilizando los estándares modernos de la industria:

* **Core:** React 18 + TypeScript
* **Build Tool:** Vite (garantizando tiempos de carga y compilación mínimos)
* **Estilos:** Tailwind CSS v4 para un diseño utility-first y 100% responsive.
* **Componentes y Accesibilidad:** Radix UI (mediante *shadcn/ui*) para componentes modulares, accesibles y altamente personalizables.
* **Despliegue (CI/CD):** Alojado en Vercel con integración directa y despliegue continuo desde GitHub.

## 🌟 Características Principales
* **Arquitectura de Componentes Limpia:** Estructura de carpetas escalable separando vistas, componentes UI genéricos (`src/app/components/ui/`) y secciones de la página (`Hero`, `TechStack`, `MainProject`, etc.).
* **Diseño Responsivo e Inmersivo:** Adaptación perfecta a dispositivos móviles, tablets y escritorio, con interacciones fluidas.
* **Optimización de Assets:** Uso de imágenes optimizadas y tipografías eficientes para evitar bloqueos en el renderizado.
